## Data Format

Each CSV file contains the following columns:
- icsad_ID
- Original_Release_Date
- Last_Updated
- Year
- ICS-CERT_Number
- ICS-CERT_Advisory_Title
- Vendor
- Product
- Products_Affected
- CVE_Number
- Cumulative_CVSS
- CVSS_Severity
- CWE_Number
- Critical_Infrastructure_Sector
- Product_Distribution
- Company_Headquarters
- License
